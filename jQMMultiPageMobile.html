This is a prototype web app for placing takeout restaurant orders.

It is mobile first -- optimized for mobile devices with jQuery Mobile.

We are using a custom jQuery Mobile theme, jQuery and Mustache.js, along with the Open Sans font from Google.

To limit the scope of this demo app, the data is mocked using JavaScript object notation instead of a database. And, CDN versions of libraries are used where possible.
